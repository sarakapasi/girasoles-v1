# Girasoles-v1-analysis
- Updates are in progress
- This is the pipeline I use for 16S and metagenomic analysis for the Girasoles initial project data w/ Kostas and Emory SON to track changes in the gut microbiome of farmworkers with varying severities of heat-related illness.
- All analysis was done on/tested in Georgia Tech's PACE Slurm system. If you have any questions, email me at sarakapasi@gatech.edu!
